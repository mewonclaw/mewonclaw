MEWCLAWING — LOGO
=================
Ganti file `logo.png` di folder ini dengan logo MEWCLAWING kamu.
Semua tempat (favicon tab, logo nav, footer, hero, CTA) otomatis pakai file ini.

Tips:
- Format PNG transparan, full color (boleh persegi atau landscape).
- Disarankan minimal 512x512 px biar tajam di semua ukuran.
- Nama file harus tetap: logo.png  (di folder assets/)
- Kalau mau favicon tab beda dari logo utama, kabari — bisa dipisah jadi favicon.png.
